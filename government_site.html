Welcome to the state government website review!

Here we will be analyzing the state website of Alaska, Delaware, Florida, Illinois, and Nebraska. Included below are a couple of metrics for each site, along with a general review, finished off with an overall ranking.

Alaska

  • Load time: quick (~1 second)
  • Logo: 6.5/10
  • Design: 8/10

'Hello, how are you?' is translated from a different language in a way that is pleasant to see. The website's margin makes the website nice to read. There are helpful drop down menus and plenty of content without being overwhelming.

Delaware

  • Load time: Slow (3-5 seconds)
  • Logo:6/10
  • Design: 2/10

Florida

  • Load time: quick (~1 second)
  • Logo: 3/10
  • Design: 3/10

I like that there is a search bar but there is no drop down when you click on it. This leaves me with little idea of what I should be searching. The content of the website is largely unknown without having explored all of the tabs (which there are not many). This website is very limited with few features. Aesthetically, it looks ok but it has little functionality.

Illinois

  • Load time: fastest (>1 second)
  • Logo: 2/10
  • Design: 4/10

A virtual assistant popped up right away which is a negative because those are never helpful. There are is some well designed helpful content, but it is a bit too busy. It could benefit from a larger margin on the main content section.

Nebraska

  • Load time: quick (~1 second)
  • Logo: 8/10
  • Design: 7.5/10

Simple, yet helpful design. Announces the important information without overwhelming the user. There is nothing that jumps out as great about this website. It does it's job, and does it pretty well.

Final State Ranking

  1. Alaska
  2. Nebraska
  3. Illinois
  4. Delaware
  5. Florida
