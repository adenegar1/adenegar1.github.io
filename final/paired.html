Paired data t-test


A paired data t-test should be conducted when data points are paired with each other. For example, this test could be used to determine if a there is a significant difference between the weights of people before and after a weight loss program.

R-code

##paired numerical data hypothesis test
#with data
c() -> a
c() -> b
-> mu
data <- a-b
xdiff <- mean(data)
sdiff <- sd(data)
ndiff <- sum((data*0)+1)
#with sample statistics
-> ndiff
-> xdiff
-> sdiff
-> mu
#calculations
SE = sdiff/sqrt(ndiff)
t = (xdiff-mu)/SE
t
#two sided p value (right)
2*(1-pt(t,ndiff-1))
#one sided p value (right)
(1-pt(t,ndiff-1))
#two sided p value (left)
2*pt(t,ndiff-1)
#one sided p value (left)
pt(t,ndiff-1)

##Paired numerical data Confidence interval
#Manually input CI value
.95 -> CI
t <- qt((1+CI)/2, ndiff-1)
SE <- sdiff/sqrt(ndiff)
MOE <- t*SE
xdiff + MOE
xdiff - MOE
