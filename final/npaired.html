Non-paired data t-test


A non-paired t-test is used to determine the likelihood of the observed difference or something more extreme between two different samples. Included below is the code for the difference of two means or the difference of two proportions.

R-code

##Difference of two means
#data entry
-> x1
-> x2
-> s1
-> s2
-> n1
-> n2
-> mudiff

#Hypothesis test
se <- sqrt((s1^2)/n1+(s2^2)/n2)
t = ((x1-x2)-(mudiff))/se
df <- if(n1 < n2) {n1-1} else {n2-1}
1 - pt(t, df)
t

##Difference of two means Confidence interval
CI <- .95
tstar <- qt((CI+1)/2, df)
(x1-x2)+se*tstar
(x1-x2)-se*tstar
MOE <- (((x1-x2)+se*tstar)-((x1-x2)-se*tstar))/2
MOE
## Difference of two proportions
#Enter Data
-> n1
-> n2
-> x1
-> x2
0 -> mudiff

#Hypothesis test Statistics
p1 = x1/n1
p2 = x2/n2
pool = (x1+x2)/(n1+n2)
hse = sqrt((pool*(1-pool)/n1)+pool*(1-pool)/n2)
t = ((p1-p2)-(mudiff))/hse
df = if(n1 < n2) {n1-1} else {n2-1}
xdiff = (x1/n1)-(x2/n2)
t

#P-Values
#two sided p value (right)
2*(1-pt(t,df))
#one sided p value (right)
(1-pt(t,df))
#two sided p value (left)
2*pt(t,df)
#one sided p value (left)
pt(t,df)

#Confidence interval
#Manually input Confidence interval value
CI <- .95
cse = sqrt((p1*(1-p1)/n1)+p2*(1-p2)/n2)
zs = qnorm((1+CI)/2)
MOE <- zs*cse
xdiff + MOE
xdiff - MOE
