Linear Regression


A linear regression is used to detirmine if a linear relationship exists between variables and gives an equation to model that relationship. Before any conclusions about linear relationship are made, a scatterplot of the data should be analyzed. In order to use the R-code for the csv input your data needs to be put into 'tidy data' format.

R-code

##Regression line (Using Array input)
X= c(2,4,4,5,6)
Y = c(5,6,7,11,12)
cor(X,Y)
fit = lm(Y ~ X)
fit
cor.test(X,Y)
summary(fit)
confint(fit)
(cor(X,Y))^2
##Regression line (Using csv input)
A = read.csv(file.choose())
head(A)
summary(A)
nrow(A)
x = A$budget
y = A$gross
plot(x,y)
abline(lm(formula=y~x))
lm(formula=y~x)
summary(lm(formula=y~x))
cor(x,y)
(cor(x,y))^2
confint(lm(formula=y~x))
