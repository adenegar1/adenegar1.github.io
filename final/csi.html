Chi Squared test of Independence


The Chi squared test of Independence is used to detirmine if respective counts are associated with each other and requires expected counts to be created.

R-code

##Chi Square test of Independence
observed <- data.frame("name" = c(1,2,3), "name2" = c(4,5,6))
df = (if(nrow(observed) == 1){1}else{(nrow(observed)-1)})*(if(ncol(observed) == 1){1}else((ncol(observed)-1)))
props <- colSums(observed)/sum(observed)
expected <- 0*observed
for (x in 1:nrow(observed)){
expected[x,] = props*rowSums(observed[x,])
}
Chi = sum((observed - expected)^2/expected)
1-pchisq(Chi, df)
