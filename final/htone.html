One sample z-test


The one sample z-test should be used when there is a given population mean and standard deviation and we are exploring whether there is evidence of a significant difference between what we observed and in our sample and the population.

R-code

##One sample z-test (Continuous)
#Enter Data
5 -> xbar
30 -> n
0 -> mu
10 -> sd
#Hypothesis test
#P-value (right)
1 - pnorm(xbar,mu,sd/sqrt(n))
#P-value (left)
pnorm(xbar,mu,sd/sqrt(n))

##One sample z-test (Proportions)
#Enter data
-> phat
-> mu
-> n

#Hypothesis test
se = sqrt((mu*(1-mu))/n)
diff = phat-mu
z = diff/se
#P-value
#two sided p value (right)
2*(1-pt(z, n-1))
#one sided p value (right)
(1-pt(z, n-1))
#two sided p value (left)
2*pt(z, n-1)
#one sided p value (left)
pt(z, n-1)

#Confidence Interval
#Manually input CI value
CI <- .95
se = sqrt((mu*(1-mu))/n)
zs = qnorm((1+CI)/2)
MOE <- zs*se
diff = phat-mu
diff + MOE
diff - MOE
