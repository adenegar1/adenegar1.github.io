Chi Squared test of Homogeneity


The Chi squared test of homogeneity is conducted when expected values are given and is used when comparing distributions of the same population.

R-code

##Chi Square test of Homogeneity
observed = data.frame(1,2,3)
expected = data.frame(4,5,6)
df = (if(nrow(observed) == 1){1}else{(nrow(observed)-1)})*(if(ncol(observed) == 1){1}else((ncol(observed)-1)))
Chi = sum((observed - expected)^2/expected)
1-pchisq(Chi, df)
