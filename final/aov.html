ANOVA


The ANOVA test or analysis of variance test is designed to compare many means and return a number of different test statistics. The usage of the ANOVA analysis cannot really be generalized in code so I've instead included an example of the usage below including a TukeyHSD analysis. To use R-code as shown, you will need to input 'tidy data'. Tidy data consists of column headers in the first row and values in the rows below. Totals should not be included in tidy data. Tidy data can be created in an excel document and saved as a csv file which can then be analyzed using this code.

The other option

Below the R-code is an ANOVA calculator that only requires that you input your data and interpret.

R-code

##ANOVA analysis
espresso = read.csv(file.choose())
boxplot(espresso$foamIndx~espresso$method)
summary(subset(espresso,method=="I-Esp"))
summary(subset(espresso,method=="Bar"))
summary(subset(espresso,method=="Hyper"))
summary(aov(espresso$foamIndx~espresso$method))
TukeyHSD(aov(espresso$foamIndx~espresso$method))
##Multi-factor ANOVA
pine = read.csv(file.choose())
summary(pine)
boxplot(pine$moisture ~ pine$months)
boxplot(pine$moisture ~ pine$package)
boxplot(pine$moisture ~ pine$solution)
summary(subset(pine,package=="glass"))
summary(aov(pine$moisture ~ pine$solution))
summary(aov(pine$moisture ~ pine$package))
summary(aov(pine$moisture ~ pine$months))
TukeyHSD(aov(pine$moisture ~ pine$solution))
TukeyHSD(aov(pine$moisture ~ pine$months))
TukeyHSD(aov(pine$moisture ~ pine$package))
summary(aov(pine$moisture ~ pine$solution*pine$package))
TukeyHSD(aov(pine$moisture ~ pine$solution*pine$package))
pine$months = as.factor(pine$months) #numerical data to categorical
